Video Demos of Code and Projects

Miami Venues - YouTube

Miami Venues - GitHub

A Ruby CLI gem application, demonstrating web scraping and object oriented programming. The CLI provides a date based search on events at the Perez Art Museum and Science Museum's Laser Fridays

Miami Rooftop Pools - YouTube

Miami Rooftop Pools - GitHub

A Sinatra web app demonstrating Sinatra based CRUD and RESTful routing. The app allows users to login with secure encryption, create a profile, add a rooftop pool and it's details, and view other user's pool collections

The Amateur Wine Reviewer - YouTube

The Amateur Wine Reviewer - GitHub

A Ruby on Rails web app demonstraing MVC architecture, CRUD and RESTful routing, several has_many through relationships, sorting and displaying data, nested routes and forms, etc. It allows users to login with secure encryption, or login with google, create a profile, add a wine and review wines in the database. Needs a new video as there are improvements.

Prototype I: A Cat's Dream - YouTube

Prototype I: A Cat's Dream - GitHub

A Ruby on Rails backend API with JavaScript front end, demonstrating usual MVC/CRUD convetions, fetch calls to add to or update the API. The user can create store game scores based on their email, update their email and start a new game. Additionally, this video demonstrates 2D video game development and the use of the Phaser3 game engine.
